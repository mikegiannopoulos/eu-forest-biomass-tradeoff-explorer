# Portfolio Translation

## CV Bullet Points

- Built a reproducible EU forest biomass scenario model in Python combining FAOSTAT land-use and forestry statistics with Eurostat GISCO spatial data to compare conservation, material cascade, and bioenergy pathways across EU countries.
- Developed a transparent carbon-accounting and robustness workflow linking forest carbon opportunity cost, biomass substitution proxies, harvest-pressure indicators, and carbon-price-based scenario ranking across Europe under 400 fixed-seed sensitivity runs.
- Extended the project to NUTS-2 level with Eurostat land-cover and crop-production data, then solved a constrained regional policy portfolio that balances forest-carbon options with food-land safeguards under explicit biomass-supply and crop-capacity conditions.

## GitHub Project Description

Reproducible EU land-use and biomass trade-off analysis using FAOSTAT and Eurostat data to compare conservation, material-use, bioenergy, and food-land safeguard priorities through carbon accounting, NUTS-2 spatial screening, sensitivity-based robustness testing, and a constrained regional portfolio model.

## Why This Strengthens Fit For The Chalmers PhD

This project creates direct evidence that the applicant can think in the way the position demands: linking land use, biomass flows, carbon accounting, spatial datasets, regional land competition, and policy-relevant scenario comparison in one reproducible workflow. It still does not overclaim to be a full integrated assessment model, but it now shows a much tighter fit with the advertised PhD because it combines forestry trade-offs, food-land competition, subnational spatial screening, official crop-production context, and a small explicit biophysical-economic allocation step.
